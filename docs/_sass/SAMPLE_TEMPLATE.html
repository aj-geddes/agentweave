Quick Start Guide

Get up and running with AgentWeave SDK in minutes. This guide walks you through installation, configuration, and creating your first secure agent.

Last updated: December 7, 2025 Reading time: 5 min

Installation

Install the AgentWeave SDK using pip. The SDK requires Python 3.8 or higher.

Bash 
pip install agentweave-sdk
Tip

For development environments, we recommend using a virtual environment to isolate dependencies.

Basic Usage

Here's a minimal example of creating a secure agent with cryptographic identity:

Python example.py 
from agentweave import Agent, Config

# Create agent configuration
config = Config(
    name="my-agent",
    workload_id="spiffe://example.org/my-agent",
    spire_socket="/tmp/spire-agent/public/api.sock"
)

# Initialize agent
agent = Agent(config)

# Define a handler
@agent.method("greet")
async def greet(context, params):
    name = params.get("name", "World")
    return {"message": f"Hello, {name}!"}

# Start the agent
if __name__ == "__main__":
    agent.serve()

API Reference

POST /rpc Stable

Execute a remote procedure call on an agent.

Request Body
method string required

The name of the method to call

params object

Parameters to pass to the method

id string

Request ID for tracking

Important Notes

Warning

Always validate SPIFFE IDs in production. The SDK enforces identity verification, but application-level authorization is your responsibility.

Security Notice

Never disable mTLS or bypass authorization checks. The SDK is designed with security-first principles and does not allow these features to be disabled.
